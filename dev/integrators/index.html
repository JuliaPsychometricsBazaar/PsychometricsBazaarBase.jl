PsychometricsBazaarBase.Integrators.CubaIntegrator — Type

CubaIntegrator is a wrapper around the Cuba.jl integration functions.

struct CubaIntegrator{AlgorithmT<:PsychometricsBazaarBase.Integrators.CubaAlgorithm, KwargsT} <: PsychometricsBazaarBase.Integrators.Integrator

  • lo::Vector{Float64}

  • hi::Vector{Float64}

  • algorithm::PsychometricsBazaarBase.Integrators.CubaAlgorithm

  • kwargs::Any

Usage example:

CubaIntegrator([0.0, 0.0], [1.0, 1.0], CubaVegas()) do x
    x[1] * x[2]
end
