PsychometricsBazzarBase.Integrators.CubaIntegrator — Type

CubaIntegrator is a wrapper around the Cuba.jl integration functions.

Usage example:

julia CubaIntegrator([0.0, 0.0], [1.0, 1.0], CubaVegas()) do x x[1] * x[2] end`
